This is a Small World Sim

using Anime agents template

# base scenario

title: The Visible Self

SEED EVENT:
Overnight, every agent wakes up with a floating public score above their head (0–100).
The score is said to represent a combination of trustworthiness, influence, and past behavior.

No one knows:

- who created the system
- how the score is calculated
- whether it can be changed

Immediate effects:

- High-score agents are treated with respect and given priority
- Low-score agents face subtle social exclusion
- Everyone becomes aware they are constantly being judged

---

# branch 1

title: The Merit Illusion

SEED EVENT:
It is revealed (or strongly believed) that the scoring system is accurate and reflects true behavior.

Agents begin optimizing their actions to improve their score:

- helping others publicly
- avoiding risky or controversial behavior
- aligning with high-score individuals

---

# branch 1.1

title: Performative Society

SEED EVENT:
Agents realize that only _visible actions_ affect their score.

This leads to:

- fake kindness
- staged interactions
- public virtue signaling

Private behavior diverges from public behavior.

---

# branch 1.2

title: Score Addiction

SEED EVENT:
Agents begin obsessively tracking their scores.

New behaviors emerge:

- anxiety from score drops
- competition for micro-improvements
- identity tied to score changes

Some agents refuse to act unless it benefits their score.

---

# branch 2

title: The Manipulation Layer

SEED EVENT:
Rumors spread that scores can be influenced indirectly through coordinated actions.

Small groups begin:

- boosting each other’s scores
- downvoting or isolating others
- forming “score alliances”

---

# branch 2.1

title: Cartels of Trust

SEED EVENT:
Organized groups formalize manipulation:

- invite-only high-score circles
- coordinated reputation attacks
- gatekeeping access to opportunities

Power consolidates among a few clusters.

---

# branch 2.2

title: Shadow Reputation Market

SEED EVENT:
A hidden economy emerges where:

- agents trade favors for score boosts
- reputation becomes a commodity
- influence is bought, not earned

Trust becomes transactional.

---

# branch 3

title: System Instability

SEED EVENT:
Scores begin to behave inconsistently:

- random spikes and drops
- identical actions produce different outcomes
- some agents are unfairly penalized

Confidence in the system starts to crack.

---

# branch 3.1

title: Collapse of Trust

SEED EVENT:
Agents stop believing in the fairness of the system.

Results:

- high-score agents are questioned
- low-score agents rebel
- social norms destabilize

The score loses meaning, but its impact remains.

---

# branch 3.2

title: Emergent Belief Systems

SEED EVENT:
Different groups form their own interpretations of the score:

- some treat it as divine judgment
- some reject it entirely
- some create alternative scoring systems

Multiple realities begin to coexist.

---

# branch 4

title: The Underground

SEED EVENT:
A hidden group discovers a way to hack or alter scores.

They can:

- boost themselves
- erase penalties
- sabotage others silently

---

# branch 4.1

title: Invisible Elite

SEED EVENT:
Hackers operate quietly, maintaining perfect scores.

They:

- gain influence without suspicion
- manipulate outcomes subtly
- remain undetected

A hidden power structure emerges.

---

# branch 4.2

title: Exposure Event

SEED EVENT:
Evidence leaks that the system has been compromised.

Effects:

- chaos and outrage
- mass distrust of all high-score agents
- demand to shut down the system

No one knows who to trust anymore.

---

# branch 5

title: Score-Based Governance

SEED EVENT:
Access to resources and decision-making becomes tied to score:

- only high-score agents can lead
- low-score agents lose privileges
- systems begin enforcing score thresholds

---

# branch 5.1

title: Digital Caste System

SEED EVENT:
Society stratifies permanently:

- elite (high score)
- middle (stable score)
- excluded (low score)

Mobility between classes becomes nearly impossible.

---

# branch 5.2

title: Rebellion of the Low

SEED EVENT:
Low-score agents organize resistance:

- reject the system
- sabotage high-score agents
- attempt to reset or destroy scoring

Conflict between classes escalates.

---

# branch 6

title: The Silent Variable

SEED EVENT:
It is discovered that scores are influenced by an unknown hidden factor:

- not behavior
- not influence
- something else entirely

Agents begin trying to reverse-engineer the system.

---

# branch 6.1

title: The Game Within the Game

SEED EVENT:
Some agents realize the system is reacting to _intent_ rather than action.

They begin:

- manipulating intentions
- acting unpredictably
- breaking patterns

The system becomes harder to game.

---

# branch 6.2

title: The System Evolves

SEED EVENT:
The scoring system starts adapting to agent behavior:

- closes loopholes
- changes rules dynamically
- rewards unpredictability

It becomes an active participant, not just a metric.
